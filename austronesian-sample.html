Proto-Samoic-outlier to Takuu

Source: [biggsHistoryPolynesianPhonology2015]

  • *f → w / # _ V *s

  • *s → h / _
  • *ʔ → ∅ / _
    • Inferred from original text: ‘loss of glottal stops’

Source: [davletshinSeeminglyOngoingSound2015]

  • *l → *r / _
    • This change did not affect geminated *ll
  • *r → l / _ l V₁ ll V₁ (r) V₁
  • *r → l / # _ a
  • *r → l / # _ o
  • *r → l / {i, e} _ {a, o}
  • *r → l / {i, e} _ u
  • *r → l / in younger speakers
